# Anwar Dahir Yahaya | Developer Portfolio

Welcome to the repository for my personal developer portfolio, showcasing my journey, skills, and projects as a Full-Stack Developer and AI Enthusiast.

## 🌟 Overview

This portfolio was built to demonstrate my proficiency in modern frontend development, featuring an engaging, high-end "dark mode" aesthetic heavily inspired by state-of-the-art tech platforms. The site utilizes glassmorphism, responsive CSS Grid layouts, and custom Intersection Observer scroll animations to deliver a premium user experience.

![Profile Avatar](public/anwar_profile.png)

## 🚀 Features
- **Responsive Design**: Mobile-first architecture ensuring the portfolio looks stunning on any device.
- **Dynamic Animations**: Custom `useScrollReveal` hook smoothly fades and translates elements into view as the user scrolls.
- **Glassmorphism UI**: Beautiful, frosted glass panels integrated with glowing neon-inspired micro-interactions.
- **Projects Showcase**: Direct links and detailed tech stacks for my deployed applications.
- **Interactive Experience Timeline**: A clean, vertical timeline highlighting my training with Voostech and the 3MTT program.

## 🛠️ Tech Stack
- **React 18**
- **Vite**
- **Vanilla CSS** (Custom CSS Variables & Flexbox/Grid)
- **Lucide React** (Icons)

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mranwar001/anwar-portfolio.git
   cd anwar-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser to view the application.
